# Dotfiles

## Requirements
- git
- fzf
- zsh
- riggrep
- neovim
- node
- yarn
- pip

## Configuration you'll get
- neovim
  - dein based
- tmux
- git
  - commit template
- zsh
  - zshrc


## How to Install
Run Script below
```
curl https://raw.githubusercontent.com/takesei/dotfiles/master/installer.sh > installer.sh
bash installer.sh
```
